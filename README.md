#  Front-end - Sistema de Gestão de Pedidos

Interface gráfica desenvolvida em **React** para consumir a API RESTful disponível no repositório: https://github.com/amandaconstante/sistema-pedidos. 


## 🔗 Links Importantes
* **Aplicação ao vivo (Front-end):** https://sistema-pedidos-frontend-production.up.railway.app/
* **Repositório do Back-end (Java):** https://github.com/amandaconstante/sistema-pedidos

## 🛠️ Tecnologias Utilizadas
* **React** (Vite)
* **JavaScript** (ES6+)
* **Integração de APIs** (`fetch` API nativa)
* **CSS3** (Estilização)
* **Integração Externa:** [API ViaCEP](https://viacep.com.br/) (para autocompletar endereços)

> ### **Uso de IA**
> A estruturação inicial de alguns componentes visuais e as regras de CSS (estilização) foram desenvolvidas com o auxílio de ferramentas de Inteligência Artificial.
